# Notification System

This project implements a notification system with backend APIs, logging middleware, and a simple frontend UI.

## Features
- Create notifications
- Fetch notifications with pagination/filter
- Mark notifications as read
- Priority-based notification handling
- Logging middleware integration

## Tech Stack
- Backend: Node.js (Express)
- Frontend: React
- Database: PostgreSQL (design)
- Logging: External logging API

## How to Run

### Backend
cd notification_app_be
npm install
node app.js

### Frontend
cd notification_app_fe
npm install
npm start
